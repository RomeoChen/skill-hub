## 一、项目背景

随着 AI Agent 技能（Skill）数量不断增长，需要一个统一的展示页面，让用户能够直观地浏览、搜索和了解所有可用技能，同时支持技能的安装引导与社区分享。

---

## 二、目标用户

| 用户类型 | 描述 |
|---------|------|
| 个人用户 | 希望快速找到适合自己需求的 AI 技能 |
| 开发者 | 希望了解技能规范、贡献或 fork 技能 |
| 访客 | 浏览技能生态，了解 AI Agent 能力边界 |

---

## 三、核心功能需求

### 3.1 技能展示（Skill Gallery）

- **技能卡片**：每个技能以卡片形式展示，包含：
  - 技能名称（Name）
  - 技能图标/Emoji
  - 一句话描述（Description）
  - 分类标签（Category Tags）
  - 安装数 / Star 数（可选）
  - 「查看详情」按钮

- **布局**：响应式网格布局，桌面端 3 列，平板 2 列，移动端 1 列

### 3.2 搜索与筛选

- **关键词搜索**：实时搜索技能名称和描述
- **分类筛选**：按标签过滤（如：效率工具、内容创作、数据分析、开发工具、生活助手等）
- **排序方式**：最新 / 最热 / 字母顺序

### 3.3 技能详情页

每个技能点击后展开详情，包含：

- 完整功能描述
- 使用场景示例（Use Cases）
- 安装方式 / 使用指引
- 技能配置参数说明
- 版本历史（Changelog）
- 相关技能推荐

### 3.4 技能分类体系

| 分类 | 说明 |
|------|------|
| 🛠️ 效率工具 | 日程、文件整理、邮件等 |
| ✍️ 内容创作 | 写作、文案、设计等 |
| 📊 数据分析 | 看板、报表、量化等 |
| 💻 开发工具 | MCP、代码、环境配置等 |
| 🌐 信息获取 | 搜索、新闻、论文等 |
| 🎨 创意艺术 | 算法艺术、设计、GIF 等 |
| 🤝 生活助手 | 习惯、健康、情感陪伴等 |

### 3.5 安装引导

- 一键复制安装命令
- 跳转到 ClawHub 安装页
- 展示依赖环境要求

### 3.6 个人技能管理（已安装）

- 展示当前已安装的技能列表
- 支持启用 / 禁用 / 卸载
- 显示技能版本与更新提示

## 四、非功能需求

| 需求 | 要求 |
|------|------|
| 性能 | 首屏加载 < 2s，搜索响应 < 100ms |
| 响应式 | 支持桌面、平板、移动端 |
| 可访问性 | 符合 WCAG 2.1 AA 标准 |
| SEO | 支持 SSG/SSR，技能详情页可被搜索引擎收录 |
| 主题 | 支持亮色 / 暗色模式切换 |

---

## 五、技术选型建议

| 层级 | 技术方案 |
|------|---------|
| 前端框架 | Vue 3 + TypeScript |
| 构建工具 | Vite |
| UI 组件库 | TDesign（与现有技术栈一致） |
| 样式 | Tailwind CSS |
| 数据来源 | 静态 JSON / GitHub API / ClawHub API |
| 部署 | GitHub Pages / Vercel |
| 路由 | Vue Router（Hash 模式，兼容静态部署） |

## 六、页面结构

```
skill-hub/
├── 首页（/）
│   ├── Hero Banner（标题 + 搜索框）
│   ├── 分类导航栏

```

## 七、数据结构设计

### Skill 数据模型

```typescript
interface Skill {
  id: string;              // 唯一标识，如 "arxiv-reader"
  name: string;            // 显示名称
  emoji: string;           // 图标 Emoji
  description: string;     // 一句话描述
  longDescription: string; // 详细描述（Markdown）
  categories: string[];    // 分类标签
  useCases: string[];      // 使用场景
  installCmd: string;      // 安装命令
  version: string;         // 当前版本
  author: string;          // 作者
  homepage?: string;       // 主页链接
  clawHubUrl?: string;     // ClawHub 链接
  dependencies?: string[]; // 依赖环境
  createdAt: string;       // 创建时间
  updatedAt: string;       // 更新时间
}
```
